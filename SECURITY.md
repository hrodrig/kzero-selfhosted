# Security policy

## Supported versions

Security fixes apply to the **latest** **[kzero-selfhosted](https://github.com/hrodrig/kzero-selfhosted)** **`main`** / tagged release of deployment manifests, in combination with supported **[kzero](https://github.com/hrodrig/kzero)** application releases.

## Reporting a vulnerability

Please report security vulnerabilities for the **kzero** project (including deployment docs in this repo) via **[GitHub private vulnerability reporting](https://github.com/hrodrig/kzero-selfhosted/security/advisories/new)** on **kzero-selfhosted**, or the equivalent on **kzero** if the issue is application-only — whichever matches the code you are reporting against.

Do not open public issues for undisclosed security vulnerabilities.
